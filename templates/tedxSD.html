What is Pandas?

Pandas is an open-source Python library built on top of Python. The name came from a combination of “Panel Data” and “Python Data Analysis.” And panel data is essentially multi-dimensional data that’s tracked over time.

History of Pandas

Pandas’s development started in 2008 by Wes McKinney, who wanted software that was suited for quantitative analysis, because he was working with financial data. It became open source in 2009, and in 2012, the same guy published Python for Data Analysis, a book. In 2015, Pandas was signed into sponsorship under this nonprofit organization NumFOCUS that supports open code, and Pandas is still under development.

Uses of Pandas

Pandas is used mainly for data manipulation, analysis, cleaning, and exploring. It’s helpful when dealing with large amounts of data, because that’s what it was made to deal with. You can also use it to see trends in data, like means or correlations across your data. Pandas also cleans up your data, in that it can align your data and it makes it easier to deal with missing data, like deleting rows that are empty or irrelevant.

Pandas also offers tools for reading data between different formats, like CSV and text files, Microsoft Excel, and SQLite databases.

Pandas vs. NumPy

NumPy is another open-source Python library that’s used for scientific computing, with complex math functions relating to muti-dimensional arrays. So, when do we use Pandas and when do we use NumPy?

We can decide based on the type of data that we’re dealing with. If we’re dealing with numerical data, which is data that is a number, then we’ll use NumPy. If we’re dealing with categorical or tabular data, like data you can logically put into a table, Pandas is more well-suited for that. Pandas uses series and data frames, two data structures, whereas NumPy is based more around arrays. NumPy’s multi-dimensional arrays are similar in structure to data frames, but data frames can contain multiple object types. Moreover, Pandas takes up more memory space, since it’s meant to be used with data that has hundreds of thousands of rows.

Pandas: Series

A series is a one-dimensional data structure, which is similar to a numPy array.

After importing Pandas, define a Series by passing a list of values.

s = pd.Series([3,1,4,1,5])

value 3 1 4 1 5

Pandas: DataFrame

A data frame is a two-dimensional tabular data structure. Every column of a data frame is a series.

Define a DataFrame by passing a dictionary or numPy array.

df = pd.DataFrame({ "Name": ["Mykolyk, Topher", "Hameed, Nada", "Chen, Fang Min"], "Age": [-1, 90, 17] })

default index Name Age
0 Mykolyk, Topher -1
1 Hameed, Nada 90
2 Chen, Fang Min 17

Pandas with Jupyter Notebook

As with any other python code, you can use Pandas in Jupyter Notebook. Jupyter Notebook is especially convenient when using Pandas, due to the easy data visualization. In Pandas, you can create different diagrams with your data frames, like bar plots.

A list of cool things you can do with Pandas

  • Import and export excel files (.xlsx), csv, etc.
  • Fast data visualization
  • Statistics functions, including describe()
  • Grouping data by categories (used for recommendation lists on platforms like Netflix!)
  • Easy management of numPy null values
